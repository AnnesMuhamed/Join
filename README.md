# JOIN - The Kanban Board 🗂️

JOIN is a collaborative Kanban board developed as a group project by a team of four people. Inspired by tools like Trello, the application allows users to organize tasks, manage workflows, and keep track of project progress in a structured and intuitive way.

## About the Project

JOIN is a task management application where users can register, log in, create tasks, organize workflows, and manage contacts.

The application includes different task categories such as:

- To Do
- In Progress
- Testing
- Done

Tasks can be created, edited, and moved across the board to reflect the current workflow and progress.

In addition, users can manage a contact list including:

- First name
- Last name
- Email address
- Phone number

Contacts can be edited and updated at any time.

The application also provides a summary dashboard where users can view task statistics, current progress, and an overview of completed and pending tasks.

## Features

- User registration and login system
- Task creation and editing
- Interactive Kanban board
- Task categories and workflow organization
- Contact management system
- Task progress overview in Summary
- Firebase integration for data persistence
- Responsive user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Firebase

## Technical Implementation

This project was built using HTML, CSS, and JavaScript.

User data, contacts, and task information are stored and managed through Firebase using a structured database system. Firebase is used to persist and synchronize application data.

The application includes dynamic task handling, board management, and interactive UI behavior using JavaScript.

Key concepts used include:

- DOM manipulation
- Dynamic rendering
- Event handling
- Data management
- Firebase integration
- Component-based structure
- Clean and maintainable code organization

## Team Collaboration

JOIN was developed as a collaborative team project with four developers. Working together helped improve communication, teamwork, problem-solving, and the ability to organize and develop features collectively.
